# oto

-   Download
    ```
    git clone https://github.com/ndiing/oto.git
    ```
-   Buka Folder
    ```
    cd oto
    ```
-   Jalankan
    ```
    node index.js
    ```
-   Lihat alamat di console (alamat & port mungkin berbeda)
    ```
    http://192.168.1.123:55341
    http://127.0.0.1:55341
    ```
